# 🍔 Online Food Delivery Customer Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Online Food Delivery Dataset** to understand customer demographics, purchasing behavior, and feedback. The analysis helps identify meaningful business insights that can improve customer satisfaction, delivery efficiency, and marketing strategies for an online food delivery company.

# 🎯 Project Objectives

The following tasks were performed during the analysis:

1. Load the dataset and display the first 10 records.
2. Find the total number of rows and columns.
3. Identify the data type of each feature.
4. Check for missing values.
5. Generate the statistical summary of numerical columns.
6. Separate categorical and numerical features.
7. Plot the distribution of customer ages.
8. Create a count plot for customer occupations.
9. Analyze monthly income distribution.
10. Find the top 5 educational qualifications.
11. Compare Gender across different Occupations.
12. Analyze the relationship between Marital Status and Feedback.
13. Compare Family Size across Monthly Income groups.
14. Visualize the relationship between Age and Family Size.
15. Create a correlation heatmap of numerical features.
16. Analyze Occupation, Monthly Income, and Feedback.
17. Compare the average Family Size by Educational Qualification.
18. Plot customer locations using Latitude and Longitude.
19. Identify the Pin Codes with the highest number of customers.
20. Perform complete EDA and generate business insights.

# 📈 Business Insights

After performing the analysis, the following insights were identified:

* Areas with the highest customer density should have more delivery partners and cloud kitchens.
* The occupation with the highest number of customers can be targeted with personalized offers and promotional campaigns.
* Monthly income groups can be used to design suitable pricing strategies and discount offers.
* Customer feedback can help improve food quality, delivery speed, and customer service.
* Family-size analysis can be used to introduce family meal combos and group discounts.
